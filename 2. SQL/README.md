# DataProject SQL - Lógica de Consultas

## Descripción del proyecto

Este proyecto consiste en la resolución de una serie de ejercicios de lógica SQL,
utilizando una base de datos en PostgreSQL.

La base de datos utilizada para el desarrollo ha sido una versión de **Shakila adaptada a PostgreSQL**.

## Objetivos

Los principales objetivos de este proyecto han sido:

- Crear e importar el esquema de la base de datos en PostgreSQL.
- Realizar consultas básicas sobre tablas individuales.
- Trabajar con filtros, ordenaciones y alias.
- Aplicar funciones de agregación y agrupaciones.
- Uso de joins para relacionar distintas tablas.
- Resolución mediante subconsultas.
- Crear una vista.
- Crear tablas temporales.
- Documentar correctamente todas las consultas.

## Tecnologías y herramientas utilizadas

- PostgreSQL
- DBeaver
- GitHub
